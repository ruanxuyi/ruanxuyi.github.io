Spring Cloud - Lec 4

Posted on 07-04-2018   |     |   visitors

Key Terms

  • Netflix Zuul: gateway service with various type of filters, which could be use to modify/inspect requests (token, authentication, etc)
  • PostgreSQL: one type of RMDBs
  • Heroku Redis: an in-memory key-val based database with fast read & write

Key Annotations

  • @EnableZuulProxy: turns the Gateway application into a reverse proxy that forwards relevant calls to other services
  • @Bean: manually create an instance and ask spring container to remember it so it could be retrived later. It is different to @Autowired in sense that, @Autowired automatically create an instance of the annotated class/interface and store it in spring container during the application start up. https://stackoverflow.com/questions/34172888/difference-between-bean-and-autowired

Spring Cloud - Lec 2

Posted on 06-15-2018   |     |   visitors

Key terms

  • Eureka Server: service discovery, microservices (client) register with service registory (server)
  • Single point of failure: part of system that, if it fails, will stop the entire system
  • Object-relational mapping (ORM): Map Java objects to database tables and vice versa
  • The Java Persistence API (JPA): One possible approach/standard for ORM
  • Java Database Conectivity (JDBC): an interface to communicate with DB, based on DB types (SQL, Oracle, MySQL, etc)
  • Entity: class which maps to database table should be marked with @Entity notation
  • Data access object (DAO): an interface to access and manipulate database content
  • Java-SE: Standard Edition, contains all of the libraries and APIs that any Java programmer should learn (java.lang, java.io, java.math, java.net, java.util, etc…)
  • Java-EE: Enterprise Edition, adds libraries which provide functionality to deploy fault-tolerant, distributed, multi-tier Java software
  • Heroku: cloud server, similiar to AWS
  • SourceTree: GIT virtualization application
  • Optional<T>: wrapper for class object to prevents from null pointer exception
  • Hibernate: one of JPA implementations

SE vs EE src: https://stackoverflow.com/questions/2857376/difference-between-java-se-ee-me

Key annotations

  • @EnableEurekaClient: register w/ discovery registory (only works for eureka)
  • @EnableDiscoveryClient: register w/ discovery registory (picks the implementation on CLASSPATH) https://stackoverflow.com/questions/31976236/whats-the-difference-between-enableeurekaclient-and-enablediscoveryclient
  • @GeneratedValue: primary key generatation type
  • @Entity: class to table in backend DB layer
  • @Table: allows to specify the details of the table (ex. table name)
  • @AutoWired: setter/getter not required for annotated varaible

Spring Cloud - Lec 1

Posted on 06-14-2018   |     |   visitors

Overview

  • RESTful API, HTTP request/response
  • Intro to Spring Boot, Maven, Tomcat, Microservice
  • Some Java Annotations
  • HW1: Uber backend project - Location Service

Key terms

  • HTTP request/response: communication protocal
  • Safe/Idempotent HTTP methods: snif, modifiy HTTP traffic
  • RESTful API: representational state transfer
  • Dependency injection: import dependent libraries
  • JDBC: Java Database Connectivity (able to switch DB type easily)
  • JPA: Java Persistence API, maps classes to tables
  • JMS: Java Message Service
  • ResponseEntity<Class_Type>: instead of return class object, which auto convert to JSON object, ResponseEntity is more flexible ( able to control everything in the response)
  • Maven: a popular build system for Spring boot
  • Tomcat: the default embedded server application in Jar for Spring
  • Jar vs War: whether or not server application is included in the compressed file
  • JavaServer Pages (JSP): a technology that generates dynamically content based on html
  • ResponseEntity: a wrapper for a HTTP response and, optionally, HTTP headers and a status code
  • pom.xml: manage dependencies, plugins, etc for Maven build system
  • Super POM and minimum POM: every POM extends Super POM
  • HttpStatusCode: HttpStatus.BAD_REQUEST, HttpStatus.CREATED, HttpStatus.OK
